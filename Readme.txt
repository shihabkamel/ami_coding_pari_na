*** I have used the laravel framework ***
--------------------------------
Follow the steps to run the code
--------------------------------
1. Open the folder into Visual studio code
2. Go to the '.env' file, edit the MAIL_HOST,MAIL_PORT,MAIL_USERNAME,MAIL_PASSWORD ( I have used mailtrap account for verification process)
3. Open the Xamp app and create a database named as "hello"
4. Now, open the terminal of visual studio code or open cmd file in this folder
5. Run "php artisan migrate" command line.
6. Run "php artisan serve" command line and wait for a link. 
7. Follow the link, which should be like this "http://127.0.0.1:8000"
